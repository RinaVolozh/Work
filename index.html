Что такое сайт?

Определение

Сайт — одна или несколько логически связанных между собой веб-страниц; также место расположения контента сервера.

  • website (english)
  • webseite (deutsch)
  • site web (français)

Обычно сайт в Интернете представляет собой массив связанных данных, имеющий уникальный адрес и воспринимаемый пользователями как единое целое. Веб-сайты называются так, потому что доступ к ним происходит по протоколу HTTP.

  1. Википедия
  2. Wikipedia (english)
  3. Wikipedia (deutsch)
